# HTML CSS JavaScript 前端三要素
软件架构
- ***B/S架构***
  - 浏览器架构
- ***C/S架构***
  - 客户端架构

***网络协议***
  - HTTP协议-明文-请求、响应模型
  - https协议-加密协议-http + ssl证书

***URL***
  - 通义资源定位符号，像是每家每户的地址一样，URL（网页的地址）是定位互联网资源的标志。
  - 格式：协议://域名[:端口号]/路径/文件名?参数
    
    协议://ip:port/path/index.html?name=value

***前后端分离开发***
- jsp  
  - 之前的开发模式 前后端代码耦合在一起 难以维护
- 前端：HTML CSS JavaScript
- 后端：Java
- 跨域：CORS
- 缓存：Cookie
